Node Slideshow

Node Slideshow is based on HTML5 and jQuery.

The information for each slide is contained inside section elements embedded in the slideshow HTML document. jQuery is then applied to give that document the behavior of a typical slide presentation when viewed in a web browser.

Controlling Slideshow

Viewers of the slideshow have their own personal keyboard controls.

Mr. Independence

The slideshow/ directory is completely independent from the rest of the project, so it can be zipped up and passed around as needed.

Package the Slideshow

  1. Open Config.js and ensure enableSocket is set to 'false'.
  2. Zip the slideshow directory.
$ cd node-slideshow
$ zip -r slideshow.zip slideshow/
